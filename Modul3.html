HTML adalah singkatan dari HyperText Markup Language, yang digunakan untuk membuat struktur dasar halaman web. HTML terdiri dari elemen-elemen seperti tag pembuka dan penutup yang digunakan untuk menandai bagian-bagian dari halaman web.

CSS, atau Cascading Style Sheets, adalah bahasa yang digunakan untuk mendesain dan mengatur tampilan halaman web. CSS memungkinkan kita untuk memisahkan konten dan desain halaman web, membuatnya lebih mudah untuk dikelola dan dipelihara.
